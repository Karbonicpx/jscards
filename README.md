A simple and responsive flashcard study tool built with only HTML, CSS, and JavaScript.
Users can study javascript related flashcards, reveal answers, and track their progress

Features:

- Navigate through a set of flashcards

= Reveal/hide andswer whenever you want

- Progress bar that updates as cards are completed

- Fully responsive layout for mobile and desktop

Perfect for quick study sessions or learning fundamentals step by step.
